# Календарь записей

Streamlit-приложение для управления записями клиентов через Google Calendar API.

## Возможности

- **Авторизация** через Google OAuth2
- **4 режима просмотра**: Сегодня, Неделя, Таблица, Календарь
- **Создание, редактирование, удаление** записей
- **Отметки посещения**: пришёл / не пришёл
- **Заметки** к каждой записи
- **Навигация по неделям и месяцам**
- **Поиск** по имени клиента и дате
- **Тёмная тема** с кастомным CSS
- **Общий календарь** — все залогиненные пользователи видят одни и те же записи

## Установка

1. Клонируйте репозиторий:

```bash
git clone <url>
cd calendar-booking
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Настройте Google Cloud Console (см. ниже)

4. Запустите:

```bash
streamlit run app.py
```

## Настройка Google Cloud Console

1. Перейдите в [console.cloud.google.com](https://console.cloud.google.com)
2. Создайте новый проект
3. Включите **Google Calendar API** в разделе APIs & Services
4. Перейдите в **Credentials** → **Create Credentials** → **OAuth client ID**
5. Тип: **Desktop app**
6. Скачайте JSON-ключ → переименуйте в `credentials.json` → положите в корень проекта
7. В **OAuth consent screen** добавьте свою почту как тестового пользователя
8. Создайте календарь «Записи» в [Google Calendar](https://calendar.google.com)
9. Настройте доступ: **Настройки → Поделиться** → добавьте email пользователей с правом «Изменение событий»
10. Скопируйте ID календаря (**Настройки → Интеграция календаря → ID календаря**)
11. Вставьте ID в `calendar_manager.py` → `CALENDAR_ID`

## Структура проекта


## Использование

### Режимы просмотра

| Режим | Описание |
|---|---|
| 📋 Сегодня | Записи на текущий день в виде карточек |
| 🗓️ Неделя | 7 колонок (Пн–Вс) с навигацией |
| 📊 Таблица | Все записи с поиском |
| 📅 Календарь | Сетка месяца с popover по дням |

### Отметки посещения

- ⏳ Ожидание — запись создана, клиент ещё не был
- ✅ Пришёл — клиент был на приёме
- ❌ Не пришёл — клиент не пришёл

### Записи

Каждая запись содержит:
- Имя клиента
- Телефон (необязательно)
- Услугу (Первичный прием / Повторный прием)
- Дату и время
- Длительность (30/60/90/120 минут)
- Заметку (необязательно)

## Технологии

- [Streamlit](https://streamlit.io/) — UI фреймворк
- [Google Calendar API](https://developers.google.com/calendar) — хранение данных
- [google-api-python-client](https://github.com/googleapis/google-api-python-client) — клиент для Google API
